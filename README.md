# 🌱 RASEED – AI-Powered Digital Receipt & Analytics Platform

## 🏆 Hackathon Context

This project was built for **Google Cloud Agentic AI Day 2025**, organized by **Hack2Skill & Google Cloud**.
Our team was selected as a **Finalist (Top 120 teams globally)** for the chosen problem statement.

---

## ❓ Problem Statement

Traditional receipts are paper-based, hard to manage, and generate massive environmental waste. Businesses struggle with fragmented receipt systems, while customers lack a unified, intelligent way to store, track, and analyze their purchase history.

Key challenges included:

* High paper wastage leading to environmental impact.
* Limited insights for customers and retailers from purchase data.
* Lack of standardization in digital receipt generation & sharing.
* Poor integration between retailers, payment gateways, and analytics platforms.

---

## 💡 Proposed Solution – RASEED

**RASEED** is an **AI-powered digital receipt and analytics platform** designed to replace paper receipts with smart, cloud-based digital ones.

### ✨ Features

* 📩 **Smart Digital Receipts** – auto-generated, cloud-stored, and easily shareable.
* 📊 **Analytics Dashboard** – insights into spending patterns, categories, and budgets.
* 🧠 **Agentic AI Integration** – contextual recommendations (e.g., expense optimization, savings suggestions).
* 🔌 **Retailer Integration** – easy API for stores to issue receipts digitally.
* 🌍 **Eco-Friendly Approach** – reducing paper usage & carbon footprint.
* 🔒 **Secure & Scalable** – built on Google Cloud with user privacy as a priority.

### 🌟 Uniqueness

* Uses **Agentic AI workflows** to not just store receipts but generate **intelligent insights**.
* Designed as a **scalable B2B + B2C SaaS model** with 5 revenue streams (Freemium B2C, B2B API Licensing, Ads & Affiliate Partnerships, Analytics-as-a-Service, Retail SaaS).
* Provides **interoperability** between banks, retailers, and end-users.

### 🌱 Impacts & Benefits

* Environmental sustainability by reducing paper usage.
* Financial literacy for customers via AI-driven analytics.
* Cost savings for retailers by cutting down on printing.
* Standardized receipt ecosystem to improve compliance & record keeping.

* **AI & Cloud:** Google Cloud (Vertex AI, Firestore, Cloud Functions, BigQuery)

## ⚙️ Setup & Installation

1. Clone the repo:

   ```
   git clone https://github.com/Jigisha-Baliyann/RASEED.git
   cd RASEED
   ```
   
2. Install dependencies:

   ```
   npm install   # for frontend
   pip install -r requirements.txt   # for backend
   ```
   
3. Set up environment variables (`.env`):

   ```
   GOOGLE_CLOUD_PROJECT=your_project_id
   DATABASE_URL=your_database_url
   OPENAI_API_KEY=your_api_key
   ```
   
4. Run locally:

   ```
   npm run dev        # frontend
   uvicorn app:main   # backend
   ```

## 🚀 Usage

* **Retailers** → integrate RASEED API into their POS systems to issue digital receipts.
* **Customers** → log in to their RASEED account to view, download, or analyze spending history.
* **Analytics** → dashboard provides AI-driven insights into personal finance & sustainability impact.
